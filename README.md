# Temperature Converter Website

## Introduction
The Temperature Converter Website is a simple web-based application developed using HTML, CSS, and JavaScript. It allows users to easily convert temperature values between Celsius, Fahrenheit, and Kelvin units.

This project was created as part of my Web Development Internship to demonstrate front-end development skills and JavaScript functionality.

## Objective
The main objectives of this project are:
- To design a simple and user-friendly interface.
- To implement temperature conversion using JavaScript.
- To validate user input.
- To display accurate results dynamically.
- To understand the integration of HTML, CSS, and JavaScript.

## Technologies Used
- HTML – For structuring the web page
- CSS – For styling and layout design
- JavaScript – For temperature conversion logic
- Visual Studio Code – Code editor used for development
- Live Server – Used to run the project locally
